pydms Feature List

A very lite document based management system 

 Upload pdf documents and maybe other types

 Every document has a :
  
  title  DONE
   
  real path  DONE

  single author  DONE
  
  a creation date  DONE
  
  a last modified date DONE
  
  a type {contract, past exam ...) DONE
  
  a description from the author DONE
  
  comments by the file users .... { commentID,userID,Date,Time,comment } DONE
  
  modification comments by the author { commentID,DocID,AuthorID,Date,Time,comment }  DONE
  
  download history {HistID,DocID,userID,date,time) DONE
  
  which students have read rights { studentID  } creator and admins can always see the file DONE
  
  set a time period for which the file is visible to the group { from datetime,to datetime } DONE
